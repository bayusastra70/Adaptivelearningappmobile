# AdaptiveLearning AI 🚀
An AI-powered mobile learning platform that generates personalized learning paths based on the skills you want to master. Built with **React Native (Expo)** and **FastAPI**, integrated with **Google Gemini AI**.

---

## ✨ Key Features

* **🎯 AI Learning Path Generation**: Automatically generates a structured curriculum for any skill using Google Gemini AI.
* **🧠 Contextual AI Mentor**: Stuck on a topic? Ask the AI Mentor for a simpler explanation directly within the lesson.
* **📝 Smart AI Quizzes**: Test your understanding with AI-generated multiple-choice questions before moving to the next step.
* **📊 Dynamic Progress Tracking**: Visual progress bars and floating dashboard cards to keep you motivated.
* **📱 Modern Floating UI**: A premium, clean, and intuitive user interface designed for a seamless learning experience.

## 🛠️ Tech Stack

**Frontend:**
* React Native (Expo Router)
* NativeWind (Tailwind CSS for Mobile)
* Axios for API Integration
* Lucide & Ionicons Icons

**Backend:**
* Python (FastAPI)
* PostgreSQL / SQLAlchemy
* Google Gemini Pro API (Generative AI)
* JWT Authentication
